

### Sheet management

- Insert new sheet
  - In the  home ribbon ->insert->insert sheet or in the bottom near the sheet's name, we find a + symbol that we can click there

- Renaming sheet
  - In the home ribbon -> format -> rename sheet
  - We can also double-click on the sheet name to edit the name

- Deleting sheet
  - We can delete a sheet  -> home ribbon -> delete->delete sheet -> prompt press ok


- hide sheets
  - We can hide the sheets -> right click on the sheet name and now click on hide,  it will hide the sheet.
  - If you want to unhide the sheet, right-click on the sheet
